C - Direction

Flexbox nous permet d'agencer ces éléments dans le sens que l'on veut. Avec flex-direction, on peut les positionner verticalement ou encore les inverser. flex-direction peu prendre les valeurs suivants :
-> row : organisé sur une ligne (par défaut)
-> column : organisé sur une colonne
-> row-reverse : organisé sur une ligne, mais en ordre inversé
-> column-reverse : organisé sur une colonne, mais en ordre inversé

D - Retour à la ligne

Par défaut, les blocs essaient de rester sur la même ligne.Si n'ont pas de place sufisante (cela provoque parfois des bugs de design). Avec flex-wrap vous pouvez demander à ce que les blocs aillent à la ligne lorsqu'ils n'ont plus de place. Les valeurs : -> nowrap : pas de retour à la ligne (par défaut)
-> wrap : les éléments vont à la ligne lorsque n' y a plus de place
-> wrap-reverse : les éléments vont à la ligne lorsque n' y a plus de place en sens inverse
